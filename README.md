# resto-app

Проект сайта ресторана.
Сделан с использованием фреймворка React и Redux, в целях обучения.

## Запуск проекта

1. Клонируем репозиторий
  ```
  git clone git@github.com:Buzhak/resto-app.git
  ```
2. Устанавливаем зависимости.
  ```
  npm install
  ```
3. Запускаем проект.
  ```
  npm start
  ```

* HTML
* CSS
* REACT
